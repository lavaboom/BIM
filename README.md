### **Running this project**
This project is written entirely in JavaScript. It consists of the following components:
- ReactJS frontend
- NodeJS backend

Running the backend:
```
cd backend
npm install
npm start
```

Running the frontend:
```
cd frontend
npm install
npm start
```
